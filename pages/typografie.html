Font questrial

Het lettertype dat gebruikt wordt bij drukwerk en web is het gratis google font questrial. Deze is perfect voor koppen en lichaamstekst
bij elk soort project. Het lettertype questrial heeft een moderne stijl, waardoor het bij elk medium goed leesbaar is.
De cirkelvormige rondingen van het font zorgen voor een uniciteit.

Font questrial in gebruik

Hieronder vindt u een voorbeeld tekst, deze bestaat uit twee gewichten van het font questrial.

  1. Font questrial 80pt bij 82pt regelafstand, reeksspatiëring 60, zwarte rand 0,2pt
  2. Font questrial 37pt bij 39pt regelafstand, reeksspatiëring 60
  3. Font questrial 14pt bij 16pt regelafstand, oranje rand 0,1 pt
  4. Font questrial 17pt bij 18pt regelafstand
  5. Font questrial 12pt bij 15pt regelafstand
